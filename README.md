# Грифоны Петербурга

Учебный веб-проект, посвящённый истории грифонов Санкт-Петербурга и их роли в архитектуре города.

Сайт объединяет исторический материал, интерактивную карту и описание архитектурных объектов.

## Возможности

- Интерактивная карта объектов
- Карточки с краткой информацией
- Подробное описание каждого объекта
- Адаптивный интерфейс для ПК и мобильных устройств
- Историческая справка о происхождении грифонов

## Использованные технологии

- HTML5
- CSS3
- JavaScript
- Leaflet
- OpenStreetMap
