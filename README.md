# Грифоны Петербурга

Учебный веб-проект, посвящённый истории грифонов Санкт-Петербурга, их происхождению и роли в архитектурном облике города.

Сайт объединяет исторические материалы, интерактивную карту объектов и подробное описание наиболее известных архитектурных сооружений с изображениями грифонов.

🌐 Сайт: https://griffins-spb.online

---

## Цель проекта

Создать интерактивный образовательный веб-ресурс, позволяющий изучить расположение и историю грифонов Санкт-Петербурга в удобном визуальном формате.

---

## Возможности

- Интерактивная карта расположения объектов
- Карточки с краткой исторической информацией
- Подробное описание каждого объекта
- Историческая справка о происхождении образа грифона
- Адаптивный интерфейс для ПК и мобильных устройств
- Просмотр маршрута по объектам на карте

---

## Использованные технологии

- HTML5
- CSS3
- JavaScript
- Leaflet
- OpenStreetMap
